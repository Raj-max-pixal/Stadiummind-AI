# STADIUMMIND AI - Product Brief

## Product

StadiumMind AI is an AI operations platform for live stadium matches. It gives fans, volunteers, and administrators a shared intelligence layer for crowd movement, safety, language support, and match-day service.

## Current Match Setup

- France vs England
- Spain vs Argentina

## Core Value

The platform brings operational intelligence into one web app:

- Fans get guidance, crowd status, emergency help, and nearby lodging.
- Volunteers get translation tools, fan-assist workflows, and alert routing.
- Admins get AI briefings, live metrics, crowd analytics, and emergency command tools.

## AI Capabilities

- Stadium assistant for fan questions
- Crowd prediction and gate recommendations
- Emergency triage and response planning
- Executive operations briefing
- Multilingual translation

## Production Stack

- Flutter web frontend
- Firebase Auth
- Firestore
- FastAPI backend
- Gemini AI
- Google Maps/location support
- OpenStreetMap Overpass lodging data
- WebSocket-ready live updates

## Website Publishing

Build the frontend with production Firebase and backend values using `--dart-define`, then publish `frontend/build/web`.
